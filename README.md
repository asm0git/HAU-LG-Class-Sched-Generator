# USC Class Schedule Generator

A static web app for building member schedules and exporting them as Monday–Saturday PDFs.

## Files
- `index.html` — markup
- `style.css` — styles
- `script.js` — app logic

## Deploy on Vercel
1. Push this folder to a GitHub repo.
2. In Vercel, click "New Project" and import the repo.
3. Framework preset: **Other** (static site) — no build step needed.
4. Deploy. Vercel will serve `index.html` automatically.

## Local preview
Just open `index.html` in a browser, or run a simple server:
```
npx serve .
```
